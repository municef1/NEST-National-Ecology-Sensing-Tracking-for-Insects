## 📋 상세 설치 가이드

### 필수 요구사항
- Python 3.8+
- 최소 4GB RAM
- GPU 권장 (CPU도 가능)

### 패키지별 설치
```bash

```

## 🎯 사용 방법

1. **브라우저 접속**: http://localhost:5000
2. **이미지 업로드**: 곤충 사진 드래그&드롭
3. **탐지 확인**: 자동 바운딩 박스 생성
4. **박스 조정**: 드래그로 위치/크기 수정
5. **분류 실행**: "확인 및 분류 진행" 버튼 클릭
